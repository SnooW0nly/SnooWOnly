<div align="center">

<img src="https://img.shields.io/badge/Projeto-Denúncias-111827?style=for-the-badge" />

**Denúncias** é um ecossistema completo criado para ajudar no combate a golpes na internet, permitindo que usuários denunciem golpistas de forma centralizada.

**Link do projeto:** https://denunciar.vercel.app/

**Arquitetura do projeto**

- Website
- Bot para Discord
